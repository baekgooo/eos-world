《어스름 너머의 세계》 게임 규칙서

1. 기본 형식

이 작품은 PDF 전자책 게임북이다. 독자는 본문을 읽고 선택지 또는 퍼즐 답안을 골라 다음 섹션으로 이동한다.

수동 기록지, 능력치, 인벤토리, 주사위, 랜덤 요소는 사용하지 않는다. 모든 상태 변화는 독자가 이동한 섹션과 장면 변형을 통해 처리한다.

2. 독자의 역할

독자는 이야기 속 주인공이다. 본문은 1인칭 “나”로 진행한다.

독자는 다음 행동을 수행한다.

  • 선택지를 고른다.
  • 본문 속 단서를 관찰한다.
  • 퍼즐을 푼다.
  • 인물의 말이 진실인지 판단한다.
  • 함정과 사기꾼을 피한다.
  • 때로는 이안을 믿거나 의심한다.

    • 3. 선택지 유형

    3.1 진행 선택

    독자의 성향이나 감정선을 반영하는 선택이다. 대부분 다음 장면으로 합류하지만, 대사나 얻는 단서가 조금 달라질 수 있다.

    예:

    
A. 이안에게 따진다.

B. 주변을 먼저 살핀다.

C. 사라진 문이 있던 자리를 만져 본다.

    3.2 퍼즐 게이트

    정답을 고르면 진행하고, 오답을 고르면 실패 페이지 또는 재도전 페이지로 간다.

    조건:

  • 정답 단서가 반드시 문제 전에 제시되어야 한다.
  • 오답 결과는 억지가 아니라 단서 미확인의 결과여야 한다.

    • 3.3 함정 선택

    겉보기에는 안전하거나 친절해 보이지만, 본문 속 작은 불일치를 발견해야 피할 수 있는 선택이다.

    예:

  • 친절한 역무원
  • 이름을 알려주겠다는 상인
  • 이안과 닮은 가짜 안내자

    • 3.4 상태 보존 선택

    수동 기록 없이 선택 결과를 다음 섹션 변형으로 이어간다.

    예:

  • 이안을 의심한 루트에서는 이안이 더 방어적으로 말한다.
  • 주변을 관찰한 루트에서는 작은 단서를 하나 더 얻는다.
  • 위험한 선택을 한 루트에서는 이후 선택지가 조금 좁아진다.

    • 4. 퍼즐 원칙

    4.1 좋은 퍼즐 조건

  • 문제 전에 단서가 있다.
  • 독자가 본문을 주의 깊게 읽었다면 풀 수 있다.
  • 정답이 작가 마음대로 정해진 것처럼 느껴지지 않는다.
  • 오답도 그럴듯하지만, 다시 읽으면 틀린 이유가 보인다.
  • 퍼즐이 세계관이나 주제와 연결된다.

    • 4.2 사용할 퍼즐 유형

  • 관찰 문제
  • 표식 해독
  • 시간표/규칙 해석
  • 거짓말쟁이 찾기
  • 이름/문장 조합
  • 기억 조각 맞추기
  • 세 개의 문 논리 문제
  • 가짜 안내자 판별
  • 숨은 문장/작은 글씨 발견

    • 4.3 피할 퍼즐 유형

  • 종이에 직접 그려야 하는 복잡한 미로
  • 장기간 기억해야 하는 암호
  • 수동 인벤토리가 필요한 문제
  • 외부 지식이 필요한 퀴즈
  • 단서 없이 찍어야 하는 문제

    • 5. 실패 페이지 규칙

    실패 페이지는 단순히 “틀렸다”가 아니라 짧은 배드엔딩처럼 쓴다.

    기본 구조:

  • 선택이 처음에는 맞는 듯 보인다.
  • 뒤늦게 이상한 점이 드러난다.
  • 독자가 놓친 단서가 명확해진다.
  • 함정이 닫힌다.
  • 실패 문구와 재도전 안내가 나온다.

    • 기본 실패 문구:

    
여기서 나의 모험은 끝났다.

다시 도전하려면 이전 페이지로 돌아가라.

    단, 모든 실패가 잔혹할 필요는 없다. 어떤 실패는 길을 잃거나, 기억을 빼앗기거나, 잘못된 세계에 갇히는 식으로 처리한다.

    6. 사기꾼/함정 인물 규칙

    사기꾼은 대놓고 수상하면 안 된다. 반드시 친절하거나 그럴듯한 표면을 가진다.

    구성 요소

  • 안심되는 표면
  • 작은 불일치 2~3개
  • 독자를 이용하려는 목적
  • 공정한 탈출 선택지

    • 불일치 예시

  • 이름표가 볼 때마다 바뀐다.
  • 그림자가 사람 모양이 아니다.
  • “안전하다”라는 말을 너무 반복한다.
  • 이안과 눈을 마주치지 않는다.
  • 발소리가 없다.
  • 독자가 말하지 않은 사실을 알고 있다.
  • 빛을 비추는 대신 빛을 삼킨다.

    • 7. 이안 사용 규칙

    이안은 안내자이지만 정답지는 아니다.

    해야 할 것:

  • 위험을 경고한다.
  • 세계의 일부 규칙을 알려준다.
  • 가끔 독자를 구한다.
  • 어떤 순간에는 불완전하거나 틀린 판단을 한다.
  • 자신의 비밀을 숨긴다.

    • 피해야 할 것:

  • 모든 퍼즐 답을 대신 알려준다.
  • 항상 옳다.
  • 지나치게 친절하고 안전하기만 하다.
  • 독자의 선택을 무의미하게 만든다.

    • 8. 분량 기준

    현재 목표:

  • 전체 PDF: 약 40~60쪽
  • 한 번의 완주 플레이: 약 25~35쪽
  • 섹션 수: 약 50~60개
  • 게임 요소: 약 12~16개
  • 실패 페이지: 약 8~12개
  • 주요 엔딩: 3~4개

    • 9. 엔딩 규칙

    주요 엔딩은 4개를 기준으로 한다.

    엔딩 1. 혼자 돌아온 방

    현실로 돌아오지만 이안을 두고 온 여운이 남는다.

    엔딩 2. 이름을 되찾은 소년

    이안의 진짜 이름을 찾아 이안을 구한다.

    엔딩 3. 별빛이 남은 세계

    주인공도 돌아오고 사잇별의 땅도 무너지지 않는다. 진엔딩에 가깝다.

    엔딩 4. 어스름에 남은 나

    주인공이 사잇별의 땅에 잠시 남는 비밀 엔딩이다.

    10. 공정성 체크리스트
