《어스름 너머의 세계》 섹션 목록 초안

이 문서는 전체 게임북을 섹션 단위로 나누기 위한 1차 설계안이다. 번호와 제목은 작업 중 변경될 수 있다.

전체 목표 수치

  • 전체 섹션: 50~60개 내외
  • 주요 장: 프롤로그 + 7장
  • 주요 엔딩: 4개
  • 실패 페이지: 8~12개
  • 게임 요소: 12~16개
