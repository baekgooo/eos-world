# 《어스름 너머의 세계》 문체·연출 가이드

## 1. 목적

이 문서는 《어스름 너머의 세계》의 문체, 분위기, 선택지 표현, 실패 페이지 톤, 삽화 지시 방향을 통일하기 위한 기준서다.

작업 중 장면이 많아져도 같은 작품처럼 느껴지도록 이 문서를 기준으로 삼는다.

## 2. 기본 문체

- 1인칭 현재형 또는 과거형을 자연스럽게 사용한다.
- 독자가 주인공이므로 “나는”을 사용하되, 과도하게 반복하지 않는다.
- 문장은 너무 길게 늘이지 않는다.
- 장면 묘사는 서정적으로, 게임 안내는 명확하게 쓴다.
- 감정은 직접 설명하기보다 몸의 반응과 주변 변화로 드러낸다.
- 장면이 진행될수록 배경 묘사와 심리 묘사를 지나치게 줄이지 않는다. 이 작품은 주인공과 독자의 내적 성장을 지향하므로, 현재 공간을 주인공의 심리와 연결해 인식하는 문장을 충분히 둔다.
- 단, 진행이 루즈해지지 않도록 긴 설명문보다 감각, 행동, 짧은 생각, 주변 사물의 반응을 통해 심리를 드러낸다.

## 3. 톤 키워드

좋은 방향:

- 어스름
- 신비로움
- 낯섦
- 조용한 긴장
- 아름답지만 불안함
- 오래된 약속
- 잊힌 이름
- 두려움을 들고 나아감

피할 방향:

- 과한 귀여움
- 장난감 같은 판타지
- 지나치게 밝고 명랑한 모험물
- 설명문 같은 세계관 해설
- 잔혹한 공포물
- 단순 권선징악

## 4. 대상 독자 감각

대상 독자는 어린아이보다 조금 더 큰 독자다. 따라서 다음을 지킨다.

- 독자를 과하게 보호하지 않는다.
- 위험과 실패를 실제로 느끼게 한다.
- 단, 잔혹하거나 혐오스러운 묘사는 피한다.
- 감정은 단순하지 않게 다룬다.
- 선택에는 의미 있는 결과가 있어야 한다.

## 5. 묘사 방식

### 좋은 묘사

현실의 사물이 살짝 어긋나는 느낌을 준다.

예:

```text
시계 초침은 움직이지 않았다. 그런데 초침의 그림자만 혼자 천천히 돌아가고 있었다.
```

```text
창밖의 골목은 그대로였지만, 가로등 불빛이 위가 아니라 아래에서 켜지고 있었다.
```

### 피할 묘사

너무 직접적으로 설명하지 않는다.

```text
나는 매우 무서웠다. 이곳은 정말 이상하고 위험한 세계였다.
```

대신:

```text
손바닥이 축축해졌다. 나는 문고리를 잡은 채, 손가락을 펴지도 오므리지도 못했다.
```

## 6. 이안 대사 가이드

이안은 짧고 여백 있게 말한다. 다만 모든 장면을 `주인공 질문 → 이안 단답 → 주인공이 왜? → 이안의 짧은 설명` 패턴으로 반복하지 않는다.

대화 변주 방식:

- 이안이 먼저 짧게 인정하거나 사과한다.
- 대답 대신 행동한다. 예: 열쇠를 숨김, 한 걸음 물러남, 그림자를 확인함.
- 질문에 바로 답하지 않고, 주인공이 직접 보도록 시선을 옮긴다.
- 모르는 것은 모른다고 하되, 매번 같은 문장으로 피하지 않는다.
- 침묵이 의미가 있을 때는 주변 반응이나 손동작으로 침묵의 무게를 보여준다.
- 주인공이 묻기 전에 이안이 자기 불완전함을 일부 드러낼 수 있다.

좋은 방향:

```text
“크게 쓰인 건 믿지 마.”
“여기선 이름이 먼저 사라져.”
“나도 내가 어디까지 기억하는지 몰라.”
“지금은 뛰어.”
```

피할 방향:

```text
“자, 내가 이 세계의 규칙을 전부 설명해줄게!”
“걱정 마, 나는 언제나 네 편이니까!”
“하하! 이건 아주 쉬운 문제야!”
```

## 7. 선택지 문체

선택지는 간결하고 행동 중심으로 쓴다.

좋은 예:

```text
A. 이안에게 사실을 묻는다.
B. 표지판의 작은 글씨를 살핀다.
C. 역무원의 이름표를 확인한다.
D. 아무 말 없이 한 걸음 물러선다.
```

피할 예:

```text
A. 매우 용감하게 앞으로 나아가서 모든 문제를 해결한다.
B. 너무 무서워서 아무것도 하지 못한다.
```

선택지에는 정답을 노골적으로 표시하지 않는다. 대신 본문 단서를 읽은 독자가 고를 수 있게 한다.

## 8. 퍼즐 문체

퍼즐은 세계관 안에 자연스럽게 들어가야 한다.

나쁜 방향:

```text
문제입니다. 다음 중 정답을 고르세요.
```

좋은 방향:

```text
시간표의 글자가 한 줄씩 흔들리더니, 세 문장만 남았다.
나는 그중 하나만 진짜 출발 시간을 가리킨다는 것을 알 수 있었다.
```

단, 선택지는 명확해야 한다.

```text
어느 문장이 진짜 시간표일까?
A. 별이 잠드는 시간
B. 그림자가 먼저 도착하는 시간
C. 이름이 불리지 않은 시간
D. 시계가 멈춘 시간
```

## 9. 실패 페이지 문체

실패 페이지는 짧고 강하게 쓴다.

기본 흐름:

1. 선택이 맞는 듯 보인다.
2. 작은 불일치가 드러난다.
3. 놓친 단서가 밝혀진다.
4. 함정이 닫힌다.
5. 실패 문구로 끝낸다.

기본 문구:

```text
여기서 나의 모험은 끝났다.
다시 도전하려면 이전 페이지로 돌아가라.
```

실패 페이지 예시 톤:

```text
역무원은 표를 받아 들고 환하게 웃었다.
그제야 나는 보았다. 그의 이름표는 처음부터 비어 있었다.
표 위에 적힌 글자가 천천히 그의 가슴으로 옮겨 붙었다.

“고마워. 이제 내 이름은 돌아왔네.”

승강장 바닥이 종이처럼 접히기 시작했다.

여기서 나의 모험은 끝났다.
다시 도전하려면 이전 페이지로 돌아가라.
```

## 10. 장면 길이 기준

PDF 전자책 게임북 기준으로 섹션 하나는 너무 길지 않게 유지한다.

권장:

- 짧은 선택 섹션: 300~600자
- 일반 장면 섹션: 700~1,200자
- 중요한 감정/전환 섹션: 1,200~1,800자
- 실패 페이지: 400~900자

단, 최종 PDF 페이지 감각에 따라 조정한다.

## 11. 삽화 지시 문체

삽화 지시는 원고와 따로 명확하게 기록한다.

형식:

```md
## 삽화 필요
- 코드: I001
- 위치: 프롤로그 시작
- 용도: 장 시작 삽화
- 장면: 내 방에 없던 문, 문틈의 별빛, 주인공 뒷모습
- 분위기: 현실적인 방 + 기묘한 어스름
- 금지: 과한 귀여움, 만화적 과장, 밝은 동화풍
```

## 12. 용어 표기 기준

현재 우선 사용:

- 어스름 너머의 세계
- 사잇별의 땅
- 초저녁 정거장
- 거울숲
- 숨은이름 시장
- 밤끝탑
- 어스름의 심장
- 별조각

표기 주의:

- 한자어가 많은 무거운 명칭은 피한다.
- 너무 아동틱한 이름도 피한다.
- 시적이지만 이해 가능한 이름을 우선한다.

## 13. 금지 표현 체크

다음 식의 표현은 피한다.

- “나는 하나도 무섭지 않았다.”
- “이건 꿈일 거야!”를 반복하는 전개
- “마법의 힘으로 모든 게 해결됐다.”
- “친구의 힘으로 다 이겼다.”처럼 단순한 교훈문
- “넌 선택받은 아이야”식의 낡은 설명
- 이안이 모든 문제를 해결하는 전개

## 14. 장면 작성 체크리스트

새 장면을 쓴 뒤 확인한다.

- [ ] 1인칭 몰입이 유지되는가?
- [ ] 설명보다 감각과 행동이 먼저 나오는가?
- [ ] 현재 공간/배경이 주인공의 심리와 연결되어 인식되는가?
- [ ] 주인공의 생각과 감정이 너무 짧게 생략되지 않았는가?
- [ ] 이안의 말투가 짧고 수상한가?
- [ ] 이안과의 대화가 반복적인 질문-단답-왜-설명 패턴에 갇히지 않았는가?
- [ ] 선택지가 행동 중심인가?
- [ ] 퍼즐 단서가 공정하게 심어졌는가?
- [ ] 너무 귀엽거나 유치한 표현은 없는가?
- [ ] 실패 가능성이 있다면 실패 이유가 납득되는가?
- [ ] 삽화가 필요한 장면은 기록되어 있는가?
